# image_download_wrapper

Wrapper for google_images_download

## Usage

(requirment: git)

```powershell
pip install -r requirements.txt
(or)
py -m pip install -r requirements.txt

python ./downloadsku.py -f "sku.txt" -s ">1024*768" -t "jpg"
```
